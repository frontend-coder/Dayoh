HTML5 шаблон Dayoh

Мой первый HTML-шаблон Dayoh, на котором я тестировал работу создания слайдера комментариев. Этот шаблон можно отнести в категорию микро Landing pages или подписные странички. Его можно 
использовать для создания подписной страницы для продвижения услуг эконом-класса в IT-сфере. Конечно же эта страничка подойдет для людей, которые желают потратить максимум денег и имеет возможность 
его доработать под свои требавания.

Я верстал HTML5 шаблон Dayoh только с одной целью показать заказчикам, какой шаблон они могут получить, если желают сильно сэкономить, при этом не желают использовать конструкторы сайтов или разнообразные билдеры.

Шаблон сверстан на основе бесплатного PSD-макета Dayoh, состоит из таких секций:

шапка с местом под текст-призыв к действию, кнопку, которую Вы можете привязать к ссылке на карточку оформления заказа и оплаты
блока преимуществ
блока призыва к действию
блока описания услуги
блока с описанием пакетов
слайдера клиентов с фотографиями
блока с простой формой заказа звонка
подвала с социальным блоком
Страница шаблона —  https://frontend-coder.github.io/portfolio/micro/dayoh/
